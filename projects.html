HPShape

An implementation of Word2Vec

After making a basic notebook implementing a basic $n$-gram generative model, this project was animplementation of the foundational NLP paper by Mikolov et al. The idea is that constructing a set of vectors per text category of a given vocabulary could help with text categorisation. Done with the help and supervision of Dr. Raphael Reinauer.
